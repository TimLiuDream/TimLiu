# 个人网站开发备注

## 当前状态
- 极简极客风主页已完成。
- 顶部导航已去掉 About / Contact 锚点，只保留品牌信息。
- 联系方式区只保留图标按钮。
- 页面内容区已比之前更宽，整体更舒展。
- favicon 已补上，来源于 `app/icon.svg`。
- Hero 区域已加宽，右侧卡片也更宽一点，避免内容出框。
- 加了更明显的彩蛋：
  - 点击头像会轮换隐藏文案
  - 左上角 TL mark hover 会显示 `still tinkering`
  - Hero 右侧终端感区域会跟着轮换文案变化
- 暗黑主题切换需要在 `<html>` 加 `suppressHydrationWarning`，已修复这一点。

## 设计意图
- 保持内容少而准。
- 尽量让页面看起来像一个安静、干净、可持续长内容的个人主页。
- 彩蛋要能发现，但不能喧宾夺主。

## 后续建议
- 如果以后还要加内容，优先放到后续独立页面，不要把首页塞满。
- 保持页内信息密度低，让头像、名字、简介和联系方式足够清楚就行。
